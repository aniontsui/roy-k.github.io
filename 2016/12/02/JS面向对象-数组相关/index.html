数组

数组是值的有序集合，每一个元素在数组中都有数字位置编号，也就是索引。数组中可以含有不同类型的元素，甚至是对象或其他数组

创建

  1. 字面量 var arr = [];

  2. 构造函数 var arr = new Array();

方法

检测数组

Array.isArray(arr);

arr instanceof Array;

({}).toString.apply(arr) === '[object Array]';  （稳定）

arr.constructor === Array;

转换方法

toString()     返回以逗号分隔的字符串

valueOf()    返回对象

toLocalString()    区别很小,如果数组调用这个方法,那么数组每一项都会调用这个方法

栈方法

push()        接收任意数量的参数加到尾部,**返回数组长度值**

pop()        从数组末尾移除最后一项，减少数组的length值，**返回该数组被删除的最后一项**

splice(s,len)    删除从指定位置开始指定数量的元素

队列方法

shift()        移除并**返回第一项**

unshift()    从数组前端添加任意个参数，并**返回新数组的长度**

splice(s,0,cont)    添加到任意位置

**数组旋转:**

arr.push(arr.shift()); // 将数组第一个剪切加到最后

arr.unshift(arr.pop()); // 将数组最后一个剪切加到第一个

操作方法

concat()    创建副本连接新数组        **原数组不变**

slice(s,e)        复制起始到终点的部分    **原数组不变**

splice(s,len,content)    替换对应部分,返回被替换的部分    **原数组被修改**

delete arr[i]    删除元素，数组长度不变，该位置变为undefined

arr.length -= 1    删除最后一个元素

位置方法

indexOf()

lastIndexOf()

排序

reverse()    反转    **原数组被修改**

sort(fn)    排序    **原数组被修改**

字符串化

join(separator)        返回通过分隔符连接数组每一个元素的字符串结果

迭代方法

callback：function(item, index, array) {}
thisArg 用于指定回调执行的上下文，即this

for循环

普通遍历

for in （会遍历出原型属性）

array.forEach(callback[, thisArg])

接收两个参数,一个是函数，另一个是运行该函数的作用域对象。

第一个参数函数接收三个参数 数组项的值 item，值的位置 idnex ，数组本身 array

every(callback[, thisArg]) 与 返回boolean值

测试数组中的每个元素都满足测试

some(callback[, thisArg]) 或 返回boolean值

测试数组中是否有元素满足callback测试

arr.filter(callback[, thisArg]) 过滤 生成新数组 原数组不变

收集通过函数测试callback的书组元素。

array.map(callback[, thisArg]) 映射 原数组不变

使用函数f修改每个数组的每个元素。按顺序收集f的每个返回值，并返回这个新组成的数组。

缩小方法

callback:function(previousValue, currentValue, index, array){}
initialValue:可以指定初始值，（如果指定，第一次循环中previousValue为初始值，currentValue为数组第一项）

reduce(callback,[initialValue]) 从左向右，使用函数r聚集数组的每个元素。可以可选的制定一个初始值v。 原数组不变

reduceRight(callback,[initialValue]) 从右向左 原数组不变

扩展

二维数组与稀疏数组

二维数组

数组的元素也是数组

稀疏数组

并不含有从0开始的连续索引，一般length比实际元素个数大

数组与对象

  1. 相同：

    都可以继承，数组也是对象，都可以添加属性

  2. 不同

    数组自动更新length
    数组按索引访问比对象访问快
    数组对象继承Array.prototype上的大量数组方法
